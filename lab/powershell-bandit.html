TEORIA O que é o PowerShell?
⚙️
Shell & Scripting

PowerShell é um terminal de comandos e uma linguagem de script criada pela Microsoft. Ele permite controlar o Windows inteiro via texto — sem mouse.

📦
Cmdlets

Os comandos do PowerShell chamam-se cmdlets (lê-se "command-lets"). Seguem o padrão Verbo-Substantivo, como Get-Process ou Set-Location.

🔎
Pipeline

Você pode encadear cmdlets com o símbolo | (pipe). A saída de um comando vira entrada do próximo — poderoso para filtrar dados.

📁
Sistema de Arquivos

PowerShell trata o sistema de arquivos como um drive. Comandos como cd, ls e cat funcionam como aliases nativos.

REFERÊNCIA Comandos que você vai usar
# Mostra onde você está agora PS C:\> Get-Location # ou o alias curto: PS C:\> pwd # Lista arquivos e pastas do diretório atual PS C:\> Get-ChildItem # alias: PS C:\> ls ou dir # Entra numa pasta PS C:\> Set-Location Desktop # alias: PS C:\> cd Desktop # Lê o conteúdo de um arquivo de texto PS C:\> Get-Content flag.txt # alias: PS C:\> cat flag.txt # Cria uma pasta nova PS C:\> New-Item -Name "minha-pasta" -ItemType Directory # Filtra saída por palavra-chave (igual ao grep no Linux) PS C:\> Get-ChildItem | Where-Object { $_.Name -like "*flag*" }
GUIA Passo a passo do desafio
1

Abrir o PowerShell

Pressione Win + R, digite powershell e pressione Enter. Você verá o prompt PS C:\Users\SeuNome>.

2

Navegar até o Desktop

PS C:\Users\SeuNome> cd Desktop # O prompt muda para: PS C:\Users\SeuNome\Desktop>
3

Procurar arquivos suspeitos

Liste tudo na pasta e preste atenção em arquivos .txt ou com nomes incomuns.

PS Desktop> ls Directory: C:\Users\SeuNome\Desktop Mode LastWriteTime Name ---- ------------- ---- -a--- 19/06/2026 07:00 desafio.txt -a--- 19/06/2026 07:00 instrucoes.pdf
4

Ler o conteúdo do arquivo

PS Desktop> cat desafio.txt Parabéns por chegar até aqui. A flag está escondida nas propriedades do sistema. Dica: $env:FLAG_SECRET
5

Acessar variáveis de ambiente

Variáveis de ambiente guardam informações do sistema. No PowerShell acessa-se com $env:NOME.

PS Desktop> $env:FLAG_SECRET FLAG{PS_N4T1V0_M4ST3R_2026}

💡 Você também pode listar todas as variáveis com Get-ChildItem Env:

6

Submeter a flag abaixo

Cole a flag encontrada no campo de submissão e clique em VALIDAR.

Missão: capturar a flag

Um arquivo foi escondido nas variáveis de ambiente do sistema. Siga o guia acima, localize a flag e submeta-a para completar o lab.
